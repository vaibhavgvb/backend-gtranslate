Translate WordPress Backend Dashboard with Google Language Translator.

== Description ==

Backend Google Translate plugin allows admins or backend users to translate backend to any language instantly using google translator

**Features**

* Translate WordPress Backend to any language

== Installation ==
1. Download the zip folder named wp-backend-gtranslate.zip
2. Unzip the folder and put it in the plugins directory of your wordpress installation. (wp-content/plugins).
3. Activate the plugin through the plugin window in the admin panel. You're Done!
